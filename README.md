# WUSD Stablecoin

Wrapped USD (Decentralized, trustless stablecoin basket consisting of 25% DAI, 25% USDC, 25% TUSD, 25% PAX)

## How to run

### Requirements

- Git
- Node.Js
- Truffle

Steps:

```
git clone https://github.com/WUSD-Stablecoin/wusd-contracts.git
cd wusd-contracts
npm install
```

Now run the tests:

`truffle test`
